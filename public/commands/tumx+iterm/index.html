Tumx+iterm

Tmux

to use Tmux with Iterm2, you only need to provide an extra option (-CC) for Tmux commands:

# create a new session*
tmux -CC 

# create a new tmux session*
tmux new -s <session-name>

# attach to a session*
tmux -CC a -t <session-name>
